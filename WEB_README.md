# 赛博墓碑生成器 - Web 版

一个基于 Flask 的 Web 应用，提供可视化界面帮助用户快速生成定制化的纪念页面。

## ✨ 特性

- 🎨 可视化配置界面
- 📤 拖拽上传照片和音乐
- 👁️ 实时预览功能
- 📦 一键生成并下载
- 🎵 音乐上传和配置
- 🖼️ 照片批量上传
- 📅 时间线编辑器
- 🎨 自定义主题颜色

## 📦 安装

### 前置要求

- Python 3.7 或更高版本

### 安装依赖

```bash
pip install -r requirements.txt
```

## 🚀 使用方法

### 启动应用

```bash
python app.py
```

应用将在 `http://localhost:5000` 启动。

### 使用步骤

1. **填写基本信息**
   - 纪念对象名称
   - 生卒年份
   - 副标题
   - 详细描述

2. **选择主题颜色**
   - 使用颜色选择器自定义主题颜色

3. **上传照片**
   - 点击选择照片按钮
   - 支持批量上传（最多 10 张）
   - 支持 JPG、PNG、GIF 格式

4. **编辑时间线**
   - 添加重要的人生事件
   - 点击"添加时间线事件"增加新条目
   - 可以删除不需要的条目

5. **配置音乐**
   - 勾选"启用背景音乐"
   - 上传音乐文件（MP3、WAV）
   - 可选择自动播放

6. **功能开关**
   - 启用/禁用烛光功能
   - 启用/禁用留言功能

7. **预览或生成**
   - 点击"预览"按钮查看效果
   - 点击"生成并下载"下载完整的网页文件

## 📁 项目结构

```
CyberGrave/
├── app.py                    # Flask 应用主文件
├── requirements.txt          # Python 依赖
├── templates/                # HTML 模板
│   ├── generator.html       # 生成器界面
│   └── preview.html         # 预览页面
├── static/                   # 静态资源
│   └── generator.css         # 生成器样式
├── src/                      # 源文件
│   ├── templates/           # 网页模板
│   ├── themes/              # 主题样式
│   └── assets/              # 资源文件
└── temp_uploads/            # 临时上传目录（自动生成）
```

## 🔧 配置说明

### 基本信息配置

| 字段 | 说明 | 必填 |
|------|------|------|
| 纪念对象名称 | 纪念对象的姓名 | 是 |
| 生卒年份 | 出生和逝世年份 | 是 |
| 副标题 | 页面副标题 | 否 |
| 详细描述 | 纪念对象的详细介绍 | 否 |

### 主题设置

| 字段 | 说明 | 默认值 |
|------|------|--------|
| 主题颜色 | 网页主题强调色 | #4a9eff |

### 照片上传

- 支持格式：JPG、PNG、GIF
- 最大数量：10 张
- 单张大小：不超过 5MB

### 时间线配置

| 字段 | 说明 |
|------|------|
| 年份 | 事件发生的年份 |
| 事件 | 事件的具体描述 |

### 音乐配置

| 字段 | 说明 |
|------|------|
| 启用背景音乐 | 是否播放背景音乐 |
| 音乐文件 | 支持的格式：MP3、WAV |
| 自动播放 | 页面加载时是否自动播放 |

### 功能开关

| 功能 | 说明 |
|------|------|
| 烛光功能 | 启用互动烛光功能 |
| 留言功能 | 启用访客留言功能 |

## 🌐 部署

### 本地部署

直接解压下载的 ZIP 文件，在浏览器中打开 `index.html` 即可。

### GitHub Pages

1. 将解压后的文件推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为发布源

### Netlify

1. 将解压后的文件拖拽到 Netlify 部署页面
2. 或将项目连接到 Netlify 并设置发布目录

## 🔒 安全说明

- 上传的文件仅用于生成网页，不会被永久存储
- 临时文件会在生成完成后自动清理
- 所有数据处理都在本地进行

## 🐛 常见问题

### Q: 上传文件失败怎么办？

A: 请检查：
- 文件大小是否超过限制
- 文件格式是否支持
- 网络连接是否正常

### Q: 预览功能无法使用？

A: 请确保：
- 浏览器允许弹出窗口
- 没有广告拦截器阻止
- Flask 应用正常运行

### Q: 生成的网页无法显示照片或音乐？

A: 请确保：
- 照片和音乐文件已正确上传
- 文件路径正确
- 浏览器支持相关格式

### Q: 如何修改已生成的网页？

A: 您可以：
- 重新使用生成器创建
- 直接编辑生成的 HTML 文件
- 修改配置文件后重新生成
